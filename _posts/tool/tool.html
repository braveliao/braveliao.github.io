3. idea绘制uml

  • 安装PlantUML插件
  1. 打开 IntelliJ IDEA。
  2. 在顶部菜单栏选择 File -> Settings。
  3. 在设置窗口中，选择 Plugins。
  4. 搜索 PlantUML，点击 Install 按钮进行安装。
  5. 安装完成后，重启 IntelliJ IDEA。
